# Excel-for-Data-Analytics-Retail-Customer-Transaction
Análisis de datos de retail utilizando Excel, aplicando limpieza y tratamiento de datos mediante definición de tipos de datos, validaciones, fórmulas condicionales y tablas dinámicas. Obtención de insights sobre comportamiento de clientes y transacciones, apoyando la toma de decisiones a partir de un análisis estructurado y visual.
